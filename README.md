# 🎮 Jogo da Vida (JavaScript)

## 📖 Sobre o projeto

Este é um jogo de simulação de vida desenvolvido em JavaScript para rodar no terminal.

Objetivo: Desenvolver um simulador de vida para praticar lógica de programação e aplicar novos conhecimentos em JavaScript.
O projeto é atualizado continuamente conforme aprendo novos conceitos, tornando-se cada vez mais completo.

---

##  Funcionalidades

-  Trabalhar
-  Receber salário
-  Sistema de energia
-  Dias passando
-  Comprar casas
-  Comprar veículos
-  Sistema de empregos
-  Cursos para melhorar a carreira
---

## 🛠️ Tecnologias

- JavaScript
- Node.js
- readline-sync

##  Como executar

1. Clone o repositório

```
git clone LINK_DO_REPOSITORIO
```

2. Entre na pasta

```
cd nome-do-projeto
```

3. Instale as dependências

```
npm install
```

4. Execute

```
node index.js
```
